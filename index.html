❓ FAQ
Questions fréquentes
Fichier encore crypté lors de l'installation +
Désactiver tous les DLC est une solution qui marche la grande majorité du temps. Va dans la bibliothèque Steam, clic droit sur le jeu → Propriétés → DLC, et décoche tous les DLC.
Code erreur 126 après le OnlineFix +
Cela veut dire que votre antivirus a supprimé un fichier nécessaire au fonctionnement du OnlineFix. Essaye de le désactiver temporairement, puis réinstalle les fichiers du OnlineFix.
Comment correctement installer un Online Fix ? +
Allez sur online-fix.me, puis cherchez le jeu que vous voulez. Vous aurez un fichier .rar ou .zip.

1. Clic droit sur le jeu dans Steam → Gérer → Parcourir les fichiers locaux
2. Glissez le fichier téléchargé ici puis extrayez-le
3. Vous pouvez maintenant lancer le jeu

⚠️ Note : Si ça vous met "self protection failed", le Online Fix ne marche pas. Vous pourrez uniquement jouer avec des gens qui ont le même Online Fix.
C'est quoi Denuvo ? +
Denuvo est un DRM (anti-tamper) utilisé par certains développeurs pour protéger leurs jeux du piratage. Il crash votre jeu s'il détecte que vous n'êtes pas sur une copie légale.

Des groupes de hackers comme voices38 travaillent à cracker ces protections. Si un jeu est cracké, il se retrouve généralement sur fitgirl-repacks.site ou steamrip.com.

Il existe d'autres DRM (Rockstar, Punkbuster, etc.) mais Denuvo est le plus efficace. Peu importe le jeu, vous pourrez y jouer un jour — Denuvo ne fait que retarder le piratage.
